# Pi 활용 아이디어

현재 Pi가 24시간 켜져있고, 웹 서버 + WOL + Agent 중개를 하고 있음.
여기에 더 붙일 수 있는 기능들 정리.

---

## 1. DDNS + 도메인 — 추천도: ★★★★★

**지금 문제**: 집 인터넷 IP가 가끔 바뀜 → 바뀌면 접속 불가
**해결**: Pi가 5분마다 "내 IP는 이거야"라고 자동 신고 → `mypc.duckdns.org` 같은 고정 주소 생성
**비용**: 무료
**난이도**: cron 한 줄 추가 (5분)

---

## 2. HTTPS (암호화 접속) — 추천도: ★★★★☆

**지금 문제**: PIN 입력할 때 비밀번호가 그대로 노출됨 (카페 와이파이 등에서 위험)
**해결**: 자물쇠 마크 뜨는 보안 접속으로 변경. PIN이 암호화되어 전송됨
**필요조건**: DDNS 먼저 설정해야 함
**난이도**: Caddy 설치 + 설정 파일 3줄 (30분)

---

## 3. 예약 작업 — 추천도: ★★★★☆

**예시**: "매일 새벽 3시에 PC 자동으로 켜서 백업 돌리고 다시 꺼줘"
**원리**: Pi가 시계 역할 → 정해진 시간에 WOL로 PC 깨움 → Agent에 명령 → 끝나면 hibernate
**용도**: 자동 백업, Windows 업데이트, 대용량 다운로드 예약
**난이도**: 현재 코드에 API 하나 추가 (1시간)

---

## 4. PC 모니터링 — 추천도: ★★★☆☆

**뭐하는 건지**: 모바일에서 PC 상태를 한눈에 확인 (CPU 몇%, 메모리 몇%, 디스크 남은 공간)
**용도**: "PC가 느린데 왜 느린지" 밖에서 확인, 디스크 꽉 찼는지 체크
**난이도**: Agent에 시스템 정보 수집 코드 추가 (1시간)

---

## 5. 알림 릴레이 (텔레그램) — 추천도: ★★★☆☆

**뭐하는 건지**: PC에서 뭔가 끝나면 폰에 알림이 옴
**예시**: "다운로드 완료", "빌드 성공", "디스크 90% 초과 경고"
**원리**: Agent가 Pi에 알림 → Pi가 텔레그램 봇으로 폰에 전송
**난이도**: 텔레그램 봇 생성 + 코드 추가 (1시간)

---

## 6. 스크린샷 원격 확인 — 추천도: ★★☆☆☆

**뭐하는 건지**: 모바일에서 버튼 누르면 PC 화면이 사진으로 찍혀서 옴
**용도**: "PC가 뭐 하고 있는지" 밖에서 빠르게 확인
**단점**: 실시간은 아님, 한 장씩 찍는 거라 답답할 수 있음
**난이도**: PowerShell 스크린캡처 + 이미지 전송 (2시간)

---

## 7. VPN (WireGuard) — 추천도: ★★☆☆☆

**쉽게 말하면**: 밖에서도 집 네트워크에 들어와 있는 것처럼 만들어 줌
**그러면**: PC 원격 데스크톱, NAS 접근, 프린터 사용 다 가능
**단점**: 설정이 좀 복잡하고, 모바일에 앱 설치 필요
**난이도**: 높음 (2~3시간, 네트워크 지식 필요)

---

## 8. 파일 다운로드 — 추천도: ★★☆☆☆

**뭐하는 건지**: PC에 있는 파일을 밖에서 모바일로 다운받기
**예시**: PC에서 토렌트로 받은 영화를 밖에서 폰으로 다운
**단점**: 대용량 파일은 느림, VPN이 더 범용적
**난이도**: 중간 (2시간)

---

## 9. 미디어 서버 — 추천도: ★☆☆☆☆

**뭐하는 건지**: Pi에 외장하드 꽂아서 넷플릭스 같은 개인 서버 만들기
**단점**: Pi 성능이 약해서 끊김 가능, PC로 하는 게 나음
**난이도**: 높음

---

## 10. 홈 자동화 — 추천도: ★☆☆☆☆

**뭐하는 건지**: "PC 켜면 방 조명도 같이 켜기" 같은 연동
**단점**: IoT 기기를 별도로 사야 함, 현재 필요 없으면 과잉
**난이도**: 높음

---
